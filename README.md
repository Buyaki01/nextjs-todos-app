# nextjs-todos-app

## Overview
This is a todos app.

## Technologies Used
- **NextJs**
- **React**
- **TypeScript**
- **Tailwind**

### Getting Started
To get started with this app, follow these steps:

1. Clone this repository: 
    ```bash 
    git clone git@github.com:Buyaki01/nextjs-todos-app.git
    ```

2. Open the repository: 
    ```bash 
    cd nextjs-todos-app
    ```

3. Install dependencies using: 
    ```bash 
    npm install next
    ```

4. Start the app using: 
    ```bash 
    npm run dev
    ```
